# Dental Vision — Dr. Vibha Arora

A clean, fast, and professional single-page portfolio website for **Dental Vision**, led by Dr. Vibha Arora, providing expert dental services including RCT, Implants, and Orthodontics in Tanda, Kangra, Himachal Pradesh.

🔗 **Live Demo:** [your-link-here.github.io](https://your-link-here.github.io)

---

## About

This portfolio is designed to help patients find and connect with Dental Vision clinic online. It highlights the clinic's expertise in advanced dental treatments, making it easy for patients to book appointments and get directions to the clinic in Tanda.

## Sections

- **Hero** — Introduction with clinic name and call-to-action
- **Services** — Dental treatments and specializations offered
- **About** — Clinic philosophy and Dr. Vibha Arora's expertise
- **Testimonials** — Patient reviews and feedback
- **Contact** — Address, phone, and Google Maps location
- **Footer** — Social media links and clinic info

## Services Covered

- 🦷 Root Canal Treatment (RCT)
- 🔩 Dental Implants
- 😁 Orthodontics & Braces
- ✨ Dental Tourism
- 🏥 Oral Surgeries

## Tech Stack

- Pure HTML, CSS & vanilla JavaScript
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Lucide Icons](https://unpkg.com/lucide@latest) for UI icons
- [Google Fonts](https://fonts.google.com/) for typography
- No frameworks · No build tools · No dependencies

## Deployment

This project is deployed via **GitHub Pages**.

To deploy your own copy:

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://your-username.github.io/repo-name`
